# Project Summary

## Overview
This project demonstrates text preprocessing using Python. The workflow reads text data, normalizes the text, removes punctuation, filters stop words, and summarizes token frequencies.

## My Modification
I applied the example to my own version of the project by creating:
- `text_data_sandra.txt`
- `text_preprocessing_sandra.py`
- `text_preprocessing_sandra.ipynb`

## Changes Made
I modified the preprocessing workflow by:
- changing the input file
- adding `text`, `data`, and `language` to the stop-word list
- calculating the number of unique cleaned tokens
- increasing the visualization from the top 10 to the top 15 cleaned tokens

## Results
The updated workflow reduced the number of dominant generic words and made the output focus more on meaningful tokens. The bar chart and token summaries helped show how preprocessing choices affect results.

## Insight
This project showed that small preprocessing changes can significantly affect text-analysis results. Adjusting stop words and output size improved the usefulness of the frequency analysis.
